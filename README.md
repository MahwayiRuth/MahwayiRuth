# 📝 Task Manager

A simple, installable web-based task manager built with HTML, CSS, and JavaScript.

## 🌟 Features
- Add / Delete / Mark tasks as complete
- Save tasks locally (localStorage)
- Installable PWA for mobile use
- Mobile-first responsive UI
